Nejdříve si musíme nainstalovat TypeScript globálně pomocí npm (Node Package Manager):

npm install -g typescript

Po instalaci ověřte, že TypeScript je správně nainstalován příkazem:

tsc --version

Vytvoření prvního TypeScript projektu

Pro vytvoření nového TypeScript projektu použijeme příkaz:

tsc --init

Tento příkaz vygeneruje soubor tsconfig.json, který slouží k nastavení konfigurace TypeScript kompilátoru.

Poté vytvoříme soubor index.ts a napíšeme náš první TypeScript kód:

let pozdrav: string = 'Ahoj, TypeScript!';
console.log(pozdrav);
