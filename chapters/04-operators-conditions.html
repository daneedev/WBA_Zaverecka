Aritmetické a logické operátory

TypeScript podporuje všechny standardní operátory JavaScriptu, včetně aritmetických (+, -, *, /, %), porovnávacích (==, ===, !=, !==, >, <, >=, <=) a logických (&&, ||, !).

// Aritmetické operátory
let soucet: number = 5 + 3;     // 8
let rozdil: number = 10 - 4;    // 6
let soucin: number = 3 * 4;     // 12
let podil: number = 20 / 5;     // 4
let zbytek: number = 10 % 3;    // 1

// Logické operátory
let a: boolean = true && false; // false (A zároveň)
let b: boolean = true || false; // true (Nebo)
let c: boolean = !true;         // false (Negace)

if, else, else if

Podmíněné výrazy v TypeScriptu fungují stejně jako v JavaScriptu, ale s výhodou typové kontroly.

let vek: number = 18;

if (vek >= 18) {
    console.log("Jste dospělý.");
} else if (vek >= 15) {
    console.log("Jste teenager.");
} else {
    console.log("Jste dítě.");
}

Typový systém v podmínkách

TypeScript umožňuje využít typový systém i v podmínkách, což je užitečné při práci s různými typy dat.

function zpracujHodnotu(hodnota: string | number) {
    if (typeof hodnota === "string") {
        // TypeScript nyní ví, že hodnota je string
        return hodnota.toUpperCase();
    } else {
        // TypeScript nyní ví, že hodnota je number
        return hodnota * 2;
    }
}

console.log(zpracujHodnotu("ahoj")); // "AHOJ"
console.log(zpracujHodnotu(5));      // 10
